### **ðŸš€ Step 2: Quiz System (MCQs, Score Tracking)**
- **Quizzes** are assigned **per day** to test users' learning.
- **Real-time score calculation** and **performance tracking**.

ðŸ“Œ **MongoDB Schema for Quizzes**
```javascript
const QuizSchema = new mongoose.Schema({
    lessonId: { type: mongoose.Schema.Types.ObjectId, ref: "LearningPlan" },
    questions: [
        {
            questionText: String,
            options: [String],
            correctAnswer: String
        }
    ],
    passingScore: { type: Number, default: 70 }
});

module.exports = mongoose.model("Quiz", QuizSchema);
```

ðŸ“Œ **API to Submit Quiz Answers & Calculate Score**
```javascript
router.post("/submit-quiz/:quizId", async (req, res) => {
    const quiz = await Quiz.findById(req.params.quizId);
    const { answers } = req.body;

    let score = 0;
    quiz.questions.forEach((q, index) => {
        if (q.correctAnswer === answers[index]) {
            score += 1;
        }
    });

    const percentage = (score / quiz.questions.length) * 100;
    const passed = percentage >= quiz.passingScore;

    res.json({ score, percentage, passed });
});
```
âœ… **Problem Solved:** **Tracks knowledge retention**, ensures **users learn effectively**.
